# Smart File Organizer

## What does this project do?
This project automatically organizes files in a selected folder by sorting them into subfolders based on their file extensions or custom rules. It helps keep your directories tidy and makes file management easier.

## How does it work?
The Smart File Organizer scans the chosen folder, classifies files by their extensions, and moves them into corresponding subfolders. It uses object-oriented programming principles and includes features such as:
- File classification and organization
- Logging of moved files (source, destination, time)
- Undo functionality (history of moved files)
- Error handling for conflicts and permissions
- Graphical User Interface (GUI) for folder selection and operation control

## Steps to run the project
1. Clone or download this repository.
2. Install required packages:
	- Run `pip install -r requirements.txt` in your terminal.
3. Start the application:
	- Run `python main.py` to launch the GUI.
4. Use the GUI to select the folder you want to organize and start the process.

Enjoy automated file organization!
